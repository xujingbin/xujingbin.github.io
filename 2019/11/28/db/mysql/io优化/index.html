发表于 2019-11-28 分类于 mysql ， io优化 ， 优化

IO优化

  • 使用磁盘阵列

  • 虚拟文件卷或软raid

  • 使用软链接

    1. 将一个数据库指向其他物理磁盘

2. 将MyISAM（其他存储引擎的表不支持）表的数据文件或索引文件指向其他物理磁盘

  • 禁止操作系统更新文件的atime属性

    LABEL=/home /home ext3 noatime 1 2

mount -oremount /home

  • 用裸设备存放InnoDB的共享表空间

  • 调整IO调度算法

    dmesg | grep -i scheduler

more /sys/block/sda/queue/scheduler

echo "deadline" > /sys/block/sda/queue/scheduler

  • RAID卡电池充放电问题

    查看RAID卡BBU状态/opt/MegaRAID/MegaCli/MegaCli64 -AdpBbuCmd -GetBbuStatus -aALL

查看RAID卡缓存策略/opt/MegaRAID/MegaCli/MegaCli64 -LDInfo -Lall -aALL

获取下次电池Relearn时间/opt/MegaRAID/MegaCli/MegaCli64 -fwtermlog -dsply -a0 -nolog  (Next Learn scheduled on:)

手动触发电池Relearn/opt/MegaRAID/MegaCli/MegaCli64 -AdpBbuCmd -BbuLearn -aALL

  • NUMA架构优化

    查看当前节点情况numactl --hardware

查看当前策略numactl --show
