hexo环境搭建

发表于 2019-11-27 分类于 环境搭建 ， hexo
hexo环境安装及参考文档

hexo

主要软件安装

安装git, nodejs略过，安装hexo

sudo npm install hexo-cli -g

创建博客（初始化站点）

hexo init username.github.io

安装主题

cd username.github.io
git clone https://github.com/theme-next/hexo-theme-next themes/next
基础配置：打开文件位置username.github.io/_config.yml修改几个键值对，下面把几个必须设置的列出来按需求修改，记得保存， 还有注意配置的键值之间一定要有空格。

写文章

所有基础框架都已经创建完成，接下来可以开始写你的第一篇博客了
在username.github.io/source/_posts下创建你的第一个博客吧

测试

hexo s

发布

安装hexo-deployer-git自动部署发布工具

npm install hexo-deployer-git –save

hexo new page “categories”
